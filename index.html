Atualizações

É uma linguagem de marcação utilizada na construção de páginas na Web por meio do Html.

No início pode parecer assustador, mas isso passa e com o tempo será mais fácil de interpretar os códigos-fontes.

O HTML5 trouxe a inclusão do elemento Canvas para desenhos, vídeo e áudio para reprodução multimídias e um melhor suporte para armazenamento local.

Cascading Style Sheets é um mecanismo para adicionar estilo a um documento web. O código CSS pode ser aplicado diretamente nas tags ou ficar contido dentro das tags style.

Apesar de não ser linguagem de programação existem tarefas que você pode resolver só com CSS sem JavaScript como por exemplo, CSS @support e o CSS Counters.

O CCS3 é simplesmente a versão mais atual do CSS, uma evolução. O CCS3 incorpora novos elementos para construir animações tanto em 2 como e 3 dimensões.

JavaScript é uma linguagem de programação interpretada estruturada, de script em alto nível com tipagem dinâmica fraca e multiparadigma.

JavaScript oferece suporte a expressões regulares de maneira semelhante ao Perl, que fornece uma sintaxe concisa e poderosa.

Na última versão foram adicionadas novas funcionalidades na linguagem como por exemplo, string.prototype.replaceAll() e promise.any().

Sass é uma linguagem de folhas de estilo concebida inicialmente por Hampton Catlin e desenvolvida por Natalie Weizenbaum.

O Sass é um grande aliado dos devs, pois agiliza e poupa tempo nas tarefas do dia a dia.

Atualmente o sass está na versão 3.7.4 que foi lançada em 2019.

O React é uma biblioteca front-end JavaScript de código aberto com foco em criar interfaces de usuário em páginas web e foi criado em 2011 pelo Facebook.

O React é flexível pois facilita a interface com outras bibliotecas e frameworks. Uma das suas principais características é a utilização de componentes e Hooks.

Atualmente possui a versão 17.0.2, apesar de possui uma versão 18 em alpha.

Git é um sistema de controle de versões distribuído, usado principalmente no desenvolvimento de software.

O Git foi desenvolvido por um cara bem inovador, importantíssimo no setor da tecnologia chamado Linus Torvalds. O responsável pelo desenvolvimento do kernel do Linux.

O Git atualmente está na versão 2.34.1.
